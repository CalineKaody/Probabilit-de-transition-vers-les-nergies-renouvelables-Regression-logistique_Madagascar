# Probabilité de transition vers les énergies renouvelables — Régression Logistique

## 📌 Description
Ce projet utilise un modèle de régression logistique en Python pour estimer la probabilité de transition vers les énergies renouvelables à Madagascar et dans plusieurs pays africains comparables.

Le modèle analyse différents indicateurs économiques et environnementaux :
- PIB par habitant
- Part des énergies fossiles
- Émissions de CO₂ par habitant

L’objectif est de prédire si un pays est engagé ou non dans une transition énergétique selon un seuil de 50% d’électricité renouvelable.

## 📊 Résultats obtenus

Le modèle calcule automatiquement les coefficients de régression :

- A → PIB par habitant
- B → Part fossile
- C → CO₂ par habitant

### Résultats pour Madagascar 🇲🇬

- Probabilité de transition en 2020 : **39.9%**
- Probabilité estimée en 2030 : **83.5%**

### Performance du modèle

- Précision globale : **90.9%**
